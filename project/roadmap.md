## 2026-02-27 Phase 2: 功能完善（1-3个月）

**摘要：** 提升产品完整性和易用性

**正文：**

### 2.1 智能摘要
**问题：** 长对话复制后超出 AI 上下文限制

**解决方案：**
- 自动摘要：调用 LLM API 生成对话摘要
- 长度选择：完整版 / 摘要版 / 自定义范围
- 多轮压缩：支持多次摘要保持核心信息

**技术方案：**
```typescript
interface SummaryOptions {
  mode: 'full' | 'summary' | 'lastN';
  maxLength?: number;      // 最大字符数
  useAI?: boolean;         // 是否使用 AI 摘要
  aiModel?: 'gpt-3.5' | 'local';
}
```

### 2.2 标签系统
**场景：** 用户有多类对话（工作/学习/生活）

**功能：**
- 自定义标签（颜色 + 名称）
- 多标签关联
- 按标签筛选/搜索

### 2.3 搜索功能
- 关键词搜索（标题 + 内容）
- 时间范围筛选
- 平台筛选
- 标签筛选

### 2.4 导出增强
- Markdown 格式导出
- HTML 格式（带样式）
- PDF 导出
- 批量导出（选择多个会话）

### 2.5 快捷操作
- 快捷键支持（Ctrl/Cmd + Shift + O 打开弹窗）
- 右键菜单（一键复制最近会话）
- 浮窗模式（页面内快速选择）

## 2026-02-27 Phase 4: 生态建设（6个月+）

**摘要：** 构建长期价值和商业可能

**正文：**

### 4.1 云端同步（可选服务）
**场景：** 用户有多台设备

**方案：**
- 端到端加密（用户持有密钥）
